Webbens Historia

Tim Berners-Lee föreslog ett protokoll för hur man kan dela data mellan olika datorer som heter “Information Management: A Proposal” WorldWideWeb webbläsaren skapas samt protokoll såsom URI (Universal Resource Identifier), HTML (Hyper Text Markup Language) och HTTP (Hypertext Transfer Protocol) Netscape Navigator blir den populäraste webbläsaren Livescript föds då man vill kunna ha dynamiska webbsidor i Netscape. Byter sedan namn till Javascript i december då Java var det populäraste språket under tiden CSS som står för Cascading Style Sheets skapas för att kunna enklare designa mer detaljerade webbsidor Javascript hade inte någon standard vilket gjorde att olika sökmotorer skapade olika varianter av Javascript. Detta gjorde det väldigt jobbigt för utvecklare att utveckla för då de var tvungna att göra en lösning för varje sökmotor. Det slutade med att ECMA-Script skapades som står för European Computer Manufacturers Association Script som är ett standardiserat Javascript. Amazon får över 1,000,000 registrerade kunder Sökmotorn Google Skapas Google blir den största sökmotorn över hela internet Facebook skapas under namnet Facemash på Harvard University av Mark Zuckerberg Instagram skapades för att enkelt kunna dela bilder online Javascript har varit ett program som kan skrivas utan att typdeklarera variabler och funktioner. Typescript ändrar detta och lägger till typer till språket vilket gör det enklare att utveckla större applikationer samt minska antalet fel. React ramverket skapas och kommer bli en av de mest populära webb-ramverken i världen. React används av webbutvecklare för att enkelt kunna utveckla hemsidor.
